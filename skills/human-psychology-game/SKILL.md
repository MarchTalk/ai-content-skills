# human-psychology-game

## 什么时候调用

- 内容已经写出来，但读完觉得"用户看了不会动"。
- 账号的咨询多但不成交，或者评论热但私信少。
- 需要写"戳一下"的开头、悬念、反问。
- 处理"用户说贵"「我考虑一下」「问了一圈」这类典型场景。
- business-thinking 锁定了阻力之后，需要把阻力翻译成用户的内心独白。

## 不要做什么

- 不要写成心理学论文（不要"马斯洛""峰终""认知失调"这种词）。
- 不要替用户喊口号（"我们都知道用户怕被坑"，要具体到怕被谁、怕在哪个动作上被坑）。
- 不要把所有用户简化成同一种心理，不同价位 / 不同决策角色 / 不同性别 / 不同城市的人心理不一样。
- 不要把心理刻画当成情绪渲染，重点是把不愿说出口的那句话写出来，不是煽情。
- 不要默认套某一个 IP 的语气来"戳人"，语气跟着当前账号的定位走。

## 工作流程

第一步：定位心理类型

用户在这一刻的状态是下面哪一种：

- **不信**：不认识你 / 觉得是广告 / 怕被坑。
- **不懂**：听不懂你在卖什么 / 比不出和别家的差别。
- **不急**：现在不需要 / 等等再说 / 没到那个阶段。
- **不敢**：怕选错 / 怕丢面子 / 怕家人反对 / 怕老板批评。
- **不甘**：付一样的钱想要更多 / 觉得自己应该被特别对待。
- **幻想**：希望有一个万能答案 / 希望一次性解决所有问题。
- **拖延**：反复咨询、收藏、加好友，但不下单。

一个用户可以同时有多种状态，要找出"这一刻最强的那一种"。

第二步：还原内心独白

把心理写成用户脑子里那句话，要满足三个条件：

- 是用户视角，不是创作者视角。
- 是用户不会主动说出口的（说出口的多半是借口）。
- 有具体场景和对象（"她半夜十二点对着购物车反复加了又删"，不是"她很犹豫"）。

第三步：找到锚点

让用户从"不会动"变成"被戳一下"，靠的是哪个锚点？

- **熟悉感**：你说的事我经历过 / 我家人就这样。
- **认同感**：原来不是只有我这么想 / 这话替我说出来了。
- **被看见**：你居然懂我没说出口的那部分。
- **羞耻 / 愧疚**：我之前那么想是错的 / 我之前那么做亏待了自己 / 家人。
- **解脱**：原来不用那么累 / 不用买那么贵也行。
- **警告**：再不动可能就来不及了 / 错过这次会后悔。

锚点和心理类型要匹配（"不信"的用户用警告无效，要先用熟悉感破冰）。

第四步：落到表达

把锚点翻译成具体的一句话、一个场景、一个动作。不要写"戳到她的愧疚感"，要写"那天她加班到十一点，孩子说妈妈你是不是不要我了"。

## 输出格式

```
## 心理类型
当下最强的那一种：（不信 / 不懂 / 不急 / 不敢 / 不甘 / 幻想 / 拖延）
理由：____

## 不会说出口的那句话
「____」（用户视角的内心独白，带场景）

## 锚点
（熟悉感 / 认同感 / 被看见 / 羞耻 / 解脱 / 警告）

## 表达方式
一句话 / 一个场景 / 一个动作的具体写法：____
```

## 质检标准

- 内心独白是用户视角，不是老板视角或创作者视角。
- 心理刻画有具体场景、具体人、具体动作，不是抽象情绪词。
- 锚点和心理类型匹配，没有用警告去戳"不信"的人。
- 没有用心理学术语装专业。
- 表达落到了一句具体的话或一个具体的画面，不是"要戳到用户痛点"这种空话。
- 语气跟着当前账号的定位走，没有把别的 IP 的语气和站位套上去。
