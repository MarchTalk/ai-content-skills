# platform-distribution

## 什么时候调用

- 选定一个选题后，决定主战场和副战场。
- 一条内容要跨平台分发，需要改成不同版本。
- 账号主问"我适合做哪个平台"。
- 诊断阶段发现"内容质量不差但平台选错了"。
- 写完稿子前，先用平台适配检查一遍开头、节奏、表达。

## 不要做什么

- 不要把抖音稿子原封不动搬到小红书或视频号。
- 不要把"全平台分发"当默认策略，多数账号只够撑一个主战场。
- 不要在小红书写抖音式开头（"今天告诉你一个真相"），用户会立刻划走。
- 不要在视频号上写陌生人逻辑（"刷到你算缘分"），视频号是熟人圈，要老板出场。
- 不要忽略平台合规线（医疗 / 金融 / 玄学 / 减肥效果 / 投资收益）。

## 工作流程

第一步：锁主战场

判断适合的主战场：

- **抖音**：用户决策快、客单偏低 / 中、需要陌生人流量、能接受表演式表达。代表行业：本地餐饮、快消、低价电商、咨询变现、知识付费。
- **小红书**：用户先搜索后下单、客单中 / 高、看封面和标题做决策、重收藏和保存、消费品 / 服务类强。代表行业：美妆、家居、母婴、教育、本地服务、轻奢。
- **视频号**：用户来源是朋友圈 / 群转发、老板人设有用、复购和介绍多、信任周期长。代表行业：高客单服务、B 端、本地品牌、有老客户基础的店主。

第二步：按平台改稿

**抖音版本**：

- 前 3 秒必须有钩子（冲突、反常识、人物动作、声音冲击）。
- 节奏快、口语化、可以"演"出来。
- 中间要有 1-2 个钩子防止划走（一句反转 / 一个画面切换 / 一个数字）。
- 结尾给观点 + 一个评论引导（不是要赞要关注，是给一个可以回评的问题）。
- 完播率优先，不要超过 90 秒，最佳 30-60 秒。
- 不要做课程感的总结结尾。

**小红书版本**：

- 封面是 80% 的胜负手，标题是另外 20%。
- 标题里要有搜索词（用户会搜的关键词），不是只有情绪词。
- 前 3 行（不展开就能看到的部分）要有结论 + 痛点 + 收藏理由。
- 中间要有清单、数字、对比、分点（手机端可读性）。
- 结尾给一个收藏理由 + 评论引导。
- 不要用抖音那种"嘿，今天告诉你一个真相"开头。

**视频号版本**：

- 老板 / 创始人出镜效果好，陌生人 IP 慎做。
- 开头要像在跟熟人讲（"上周有个客户跟我聊..."），不是"刷到你算缘分"。
- 节奏可以慢一点，但要真实场景、真实人、真实动作。
- 朋友圈预热和评论区回复是分发链路的一部分，不要写完视频就不管了。
- 结尾给一个动作（私信、看朋友圈、到店、看直播），不要喊点赞。

第三步：合规检查

按行业过一遍红线：

- 医疗 / 医美：不能保证疗效，不能比对效果，不能贬低同行医院。
- 教育 / 培训：不能承诺分数、不能用焦虑性表达。
- 投资 / 金融：不能写收益率、不能承诺保本。
- 减肥 / 健康：不能写极端数据、不能写"几天瘦多少斤"。
- 玄学 / 命理：不能用断言、不能涉及医疗替代。
- 本地服务：注意区域性广告法（一线城市执行更严）。

红线上的表达要在最后一步过一遍，宁可改不要赌。平台规则会更新，拿不准的去平台官方规则中心查当前原文。

第四步：分发节奏

- 主战场每周更新频率（抖音建议 3-5 条 / 周，小红书 3-5 条 / 周，视频号 1-3 条 / 周）。
- 副战场怎么用主战场的素材二次加工，不是另起炉灶。
- 朋友圈 / 群作为补充承接，不要漏掉。

## 输出格式

```
## 主战场
平台：____
理由：____

## 副战场（如有）
平台：____
作用：____（导流 / 信任 / 复购）

## 同一选题的平台版本

### 抖音版本
开头：____
钩子：____
结尾：____

### 小红书版本
封面方向：____
标题（含搜索词）：____
前 3 行：____
中段结构：____
结尾：____

### 视频号版本
谁出镜：____
开头（熟人语气）：____
朋友圈接力：____
结尾动作：____

## 合规检查
行业红线：____
本稿风险点：____
处理方式：____
```

## 质检标准

- 主战场选择有依据（不是"反正都做一下"）。
- 三个平台的版本写法明显不同，不是同一稿改头换尾。
- 抖音版本前 3 秒有具体钩子（写得出来的，不是"要有冲击力"）。
- 小红书版本的标题里能数出搜索词。
- 视频号版本不像在跟陌生人讲。
- 合规风险点已逐条核对，不在红线上。
