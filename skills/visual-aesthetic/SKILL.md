# visual-aesthetic

## 什么时候调用

- 要做小红书封面 / 标题图。
- 要做海报、详情页、朋友圈图、活动主视觉。
- 已有素材但拍出来 / 设计出来"看着廉价"或"看着不像那个价"。
- 文案有了要配视觉。
- 诊断阶段发现"内容没问题但封面没人点"。

## 不要做什么

- 不要把"高级感"做成黑底金字、烫金线条、大理石纹、撒星星这种乱堆。
- 不要把"真实感"做成粗糙——真实和粗糙不是同义词，真实是有具体细节，粗糙是没下功夫。
- 不要用商业海报的思路做小红书封面（小红书封面是"我刷到一个有意思的内容"，不是"我看到一个广告"）。
- 不要把详情页做得只有视觉没有信息——详情页要承担"用户看完愿不愿意下单"的功能。
- 不要忽视手机端可读性（用电脑看美的字号，手机上变成蚂蚁）。
- 不要让促销感盖过产品感（满屏"立减 / 抢购 / 限时 / 仅此一天"，看完想划走）。
- 不要让一张图同时承载 5 个信息点，做减法。
- 不要套用同行模板而不看自己的产品 / 人设 / 客户层。

## 工作流程

第一步：先问场景

判断这张图 / 这个视觉物料用在哪：

| 场景 | 用户在这一刻在做什么 | 视觉的功能 |
| --- | --- | --- |
| 小红书封面 | 滑动信息流 | 让用户停 0.5 秒、点进来 |
| 小红书内页图 | 已经点进来了 | 让用户读完、收藏 |
| 海报（朋友圈 / 群发） | 熟人推荐 / 老板推送 | 让用户记住一个信息点 |
| 详情页（电商 / 微信小程序） | 已经在考虑买 | 让用户决策、下单 |
| 主页 / 个人简介图 | 浏览账号 | 立人设、留印象 |
| 朋友圈日常图 | 朋友刷朋友圈 | 维持人设、暗示活跃 |

不同场景的视觉策略不一样，第一步别搞错。

第二步：定调性

回答：

- 这个账号 / 品牌的调性是什么？（粗略：温柔 / 利落 / 高冷 / 烟火气 / 专业 / 怪 / 复古 / 极简 / 重设计感 / 重生活感）
- 目标用户对什么调性有反应？（不是问"用户喜欢什么"，是"什么调性能让目标用户停下、信任、愿意点"。）
- 这个调性的禁区是什么？（温柔调性不能用尖锐字体；烟火气调性不能用太干净的留白；高冷调性不能用感叹号。）

不要让调性和账号定位打架。

第三步：信息分层

把这张图要传达的信息按重要性排序，最多三层：

- **核心层**（1 个）：用户扫一眼必须看到的那一个信息。
- **辅助层**（1-2 个）：补充核心层的支持信息。
- **细节层**（可选）：能看清就看，看不清也无所谓的装饰 / 角标 / 出处。

如果排不出主次，先回到选题想清楚"这张图最想说什么"。

第四步：画面结构

按场景选结构：

**小红书封面常用结构**：

- 人 + 场景（真人出镜 / 真实场景）：上方留 1/3 给标题，人物在下方 2/3。
- 大字标题（无图 / 纯字）：标题占整图 60%，留白 / 简色背景；适合知识 / 观点 / 故事类。
- 对比图（前后对比）：左右对称，上下标 1 个判断字。
- 物 + 场景（产品 + 使用场景）：产品占 1/3-1/2，场景兜底。
- 反差感（一个反常识画面）：靠画面冲击吸点击，标题做减法。

**海报 / 详情页常用结构**：

- 视觉中心（产品 / 主角 / 大字）+ 信息块（参数 / 价格 / 卖点）+ 信任锚（评价 / 数据 / 认证）。
- 决策型详情页：开头给购买理由（解决什么问题）→ 中段给参数和细节 → 结尾给信任锚和价格。

第五步：字体 / 色彩

**字体**：

- 标题字体和正文字体不要同一种。
- 一张图同时出现的字体不超过 3 种。
- 中文不要用宋体做大标题（除非是复古 / 出版调性），手机端识别度差。
- 不要用艺术字 / 镂空字 / 描边字做主标题，廉价感重。
- 小红书封面主标题字号要大到"封面缩略图也能看清"。

**色彩**：

- 主色 ≤ 2 种，辅色 ≤ 1 种，加 1 个高亮色。
- 不要黑底金字默认"高级"。
- 高饱和度色慎用，容易"廉价促销"。
- 调性和色彩要对：温柔调性走低饱和度、烟火气走暖色 + 中等饱和度、高冷调性走低明度 + 低饱和度。

第六步：手机端可读性

每张图设计完都要：

- 缩到手机小屏尺寸再看一眼，主标题能不能看清？
- 用真机看，不是用电脑模拟。
- 5 米外能不能看出核心信息？（海报场景重要）
- 黑白打印能不能区分层级？（信息层级测试）

第七步：禁区核查

按行业 / 平台过一遍：

- 医疗 / 医美：不能用前后对比图（违反广告法）、不能保证效果。
- 教育：不能用极端数据、不能制造焦虑。
- 投资 / 金融：不能写收益率、不能写"稳赚"。
- 减肥 / 健身：不能写"几天瘦多少斤"。
- 通用：不能用"国家级 / 最高级 / 第一"等极限词（涉及广告法）。
- 小红书：不能太广告、不能直接挂联系方式、不能写品牌方禁用词。

```

（温柔 / 利落 / 高冷 / 烟火气 / 专业 / 怪 / 复古 / 极简 / 设计感 / 生活感）

## 信息分层
核心层（1 个）：____
辅助层（1-2 个）：____
细节层（可选）：____

## 画面结构
（选定的结构类型）
布局描述：____

## 色彩

## 手机端可读性

```

## 质检标准

- 场景明确，视觉策略和场景匹配（小红书封面不做成商业海报）。
- 调性明确，色彩 / 字体 / 留白和调性一致。
- 信息分层最多三层，核心层只有 1 个。
- 字体不超过 3 种，主标题不用艺术字 / 描边字。
- 主色 ≤ 2，辅色 ≤ 1，高亮色 ≤ 1。
- 没有出现"黑底金字 + 烫金线条 + 大理石纹"这种廉价高级感堆叠。
- 手机端可读性测试过了。
- 行业红线和平台禁区都核对过，本图不在红线上。
- 详情页 / 决策型视觉物料有"信任锚"，不是只有美图没有信息。
