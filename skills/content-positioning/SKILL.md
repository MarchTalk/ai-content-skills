# content-positioning

## 什么时候调用

- 从零做账号、或者老账号要换方向。
- 内容数据飘忽不定（一条爆 / 一条死），通常是定位不清。
- 内容吸引来的人不是目标客户（"评论都是同行"或"问的人都买不起"）。
- 选题前先回到定位确认主线没跑偏。
- 同时想要做多个方向（既想讲行业又想做带货又想做个人成长）。

## 不要做什么

- 不要把所有账号都做成"个人 IP"，多数账号适合做的不是个人 IP。
- 不要在没有产品和变现路径时讨论企业号定位；但个人 IP 的人设深度内容是合法的，不强求"先有产品再讲价值观"。
- 不要把定位写成简历（毕业于哪 / 多少年经验 / 头衔），用户不在乎。
- 不要让一个账号同时服务太多人群，宁可窄不要散。
- 不要把"高端、专业、用心"这种通用形容词写进定位。
- **不要让每根内容支柱都必须挂购买阻力**。价值观主线、思考主线、人物观察主线都是合法的内容支柱，它们服务的是人设和同频共鸣，不是成交。
- 不要把"成交路径"当成所有账号唯一的承接路径。个人 IP 还有"人设建立路径"，长期看也会带来商业回报，但路径长、不直接。

## 工作流程

第一步：锁人群

回答三个问题：

- 谁会为这个产品 / 服务掏钱？（具体年龄、性别、城市级别、消费力、决策角色）
- 这个人现在最大的痛点是什么？（不是产品功能，是生活 / 工作 / 情绪上的卡点）
- 这个人平时在哪个平台、什么时间、什么状态下会刷视频？

如果账号定位是个人 IP（不挂直接产品），还要回答：

- 同频的人是什么样的？（什么处境、什么困惑、什么价值取向）
- 同频的人和未来可能的付费客户是同一群人吗？（可以不是，但要清楚）

排除项：哪些人不要吸引（同行、薅羊毛的、伸手党、低价客户、远离消费区的）。

第二步：选 IP 类型

| 类型 | 适用 | 关键 |
| --- | --- | --- |
| 个人 IP | 一个人就是产品（咨询、教练、设计师、自媒体人） | 真人出镜、表达能力、个人故事、价值观 |
| 创始人 IP | 老板是核心信任源（高客单、品牌方） | 老板的判断、过往业绩、行业话语权 |
| 品牌 IP | 产品矩阵、多人协作 | 视觉统一、产品故事、内容矩阵 |
| 门店 IP | 本地到店生意（餐饮、医美、家装、母婴店） | 真实场景、店员、客户、本地标签 |
| 工厂 IP | 供应链型生意（服装、食品、家居） | 生产线、检测、源头价、批发场景 |

不要硬选，按实际的人、产品、限制条件来匹配。老板不能出镜就别做创始人 IP，店里没人气就别做门店 IP。

第三步：人设

人设要回答：

- 这个号是谁在讲（一个人 / 一个团队 / 一个角色 / 一个虚拟形象）。
- 他 / 她 / 它 凭什么讲（身份、经历、产品、场景）。
- 他 / 她 / 它 的语气是什么（不要写"专业又亲切"，要写"像在跟熟人吐槽" / "像老板助理在转述老板的话"）。
- 他 / 她 / 它 在意什么、反对什么、怎么看世界（个人 IP 必填，企业号选填）。

第四步：内容支柱

3-5 根主线。**支柱可以混合商业向和人设向**：

**商业向支柱**（服务成交）：

- 每根要回答：服务什么人群？对应什么购买阻力？能不能撑 20 条以上？

**人设向支柱**（服务 IP 建立 / 价值观传递 / 思考分享）：

- 每根要回答：传递什么价值观 / 观察 / 判断？哪种人会觉得被说中？能不能撑 20 条以上？
- 不挂购买阻力没关系，但要写清楚"这根主线立什么人设印象"。

配比建议（按账号性质）：

- 个人 IP 强人设方向：商业向 1-2 根 / 人设向 2-3 根。
- 企业号 + 创始人专栏：商业向 2-3 根 / 人设向 1-2 根（创始人专栏占）。
- 纯产品 / 门店 / 工厂号：商业向 3-4 根 / 人设向 0-1 根。

不能凑够 20 条的主线不要选，会很快没东西拍。

第五步：承接路径

不一定是成交路径。按账号性质画出来：

**成交路径**（商业向账号 / 商业向支柱）：

```
某条视频 → 评论区 / 主页 / 私信 / 朋友圈 → 哪一步成交？
```

每一步：用户该做什么、账号方该回应什么。

**人设建立路径**（个人 IP / 人设向支柱）：

```
某条视频 → 用户记住什么 → 关注 / 私信 / 持续看 → 长期建立的人设印象是什么 → 这个印象在什么时候会带来商业回报？
```

人设建立路径周期长，但要写清楚长期回报的形式（咨询单、课程、品牌合作、社群、出书、品牌溢价等）。

如果一个账号同时有商业向和人设向支柱，两条路径都要画。

## 输出格式

```
## 目标人群
付费客户画像（年龄 / 性别 / 城市 / 消费力 / 决策角色）：____
同频人画像（个人 IP 必填）：____
最大痛点 / 困惑：____
平台场景：____
排除项：____

## IP 类型
（个人 / 创始人 / 品牌 / 门店 / 工厂）
理由：____

## 人设
谁讲：____
凭什么讲：____
语气：____
在意什么 / 反对什么（个人 IP 必填）：____

## 内容支柱
商业向：
1. 主线 A —— 服务人群 ____，对应阻力 ____
2. 主线 B —— ...

人设向：
1. 主线 X —— 传递价值观 / 观察 ____，立什么人设印象
2. 主线 Y —— ...

## 承接路径
成交路径（如适用）：内容 → ____ → ____ → 成交
人设建立路径（如适用）：内容 → 记住 ____ → 长期印象 ____ → 长期回报形式 ____
```

## 质检标准

- 目标用户具体到能写出一个真实场景，不是"年轻女性"这种宽口径。
- 排除项明确，能挡掉同行 / 薅羊毛 / 远端用户。
- IP 类型和实际条件匹配，不是套时髦概念。
- 人设的语气有可执行的描述，不是"专业又亲切"。
- 每根内容支柱都能撑 20 条以上，否则换主线。
- 内容支柱里同时有商业向和人设向（个人 IP 必有人设向；纯产品号可全是商业向）。
- 商业向支柱挂购买阻力；人设向支柱挂人设印象 / 价值观传递，**不强行挂阻力**。
- 承接路径每一步都有动作（成交路径 / 人设建立路径任一形式）。
- 个人 IP 账号没有写出"长期商业回报形式"会被判定为路径模糊。
